# Installation and running

## Prerequisites
* SQLite3
* Python 3.6

Clone repository: 
```
git clone https://github.com/k0tix/tsoha2019.git
cd tsoha2019
```

Initialize python venv:
```
python3 -m venv venv
source venv/bin/activate
```

Install requirements:
```
pip install -r requirements.txt
```

Run application:
```
python3 run.py
```
